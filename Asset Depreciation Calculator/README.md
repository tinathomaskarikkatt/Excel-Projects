# 📊 Asset Depreciation Analysis & Calculator

This Microsoft Excel project automates asset depreciation calculations using two widely used accounting methods: **Straight Line Method (SLM)** and **Diminishing Balance Method (DBM)**. It generates depreciation schedules, tracks book values over an asset's useful life, and presents the results through analysis tables and charts.

---

## 📌 Project Overview

Asset depreciation is an important financial process that helps determine how an asset loses value over time. This project simplifies depreciation calculations by allowing users to enter asset details and automatically generating depreciation schedules using different methods.

The workbook also includes analysis sheets and a dashboard to compare depreciation trends and book values.

---

## 🎯 Objectives

- Compare Straight Line Method (SLM) and Diminishing Balance Method (DBM).
- Generate year-wise depreciation schedules.
- Track asset book values throughout their useful life.
- Present depreciation trends using charts and dashboards.

## 📊 Depreciation Methods Implemented

### Straight Line Method (SLM)

- Calculates an equal depreciation amount each year.
- Suitable for assets that depreciate uniformly over time.

### Diminishing Balance Method (DBM)

- Applies a fixed depreciation rate to the remaining book value.
- Results in higher depreciation during the initial years and lower depreciation in later years.
