# 📂 Projects Included

### 📊 [Style Advisor Performance Analysis](./Style%20Advisor%20Performance%20Analysis)


An Excel-based data analysis project that evaluates the performance of Style Advisors using customer evaluation data. The project analyzes regional performance, categorizes advisors based on evaluation scores, and presents insights through Pivot Tables, Pivot Charts, and an interactive dashboard.

### Highlights

- Performance grading
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Business Insights
- Regional Performance Analysis

---

### 💰 [Asset Depreciation Calculator](./Asset%20Depreciation%20Calculator)

An Excel-based financial analysis project that automates asset depreciation calculations using both the **Straight Line Method (SLM)** and **Diminishing Balance Method (DBM)**.

The workbook generates depreciation schedules, tracks book values, and visualizes depreciation trends through charts and dashboards.

### Highlights

- Financial Analysis
- Excel Formula Automation
- Depreciation Calculator
- Dashboard
- Comparative Analysis
- Book Value Tracking
